Day 7 HR Attendance Analysis

Project Overview
Analyzed employee attendance data for 20 employees across 5 departments (IT, HR, Sales, Finance, Marketing). Created dashboard tracking attendance percentage, punctuality scores, and performance categorization.

Key Features
- Department-wise attendance analysis
- Designation-wise attendance comparison
- Performance category distribution (Excellent, Good, Average)
- Late arrivals tracking by department
- Absent days analysis by department
- Top 5 employees by attendance
- KPI cards for key metrics
- Conditional formatting for data visualization

Formulas Used

Attendance Percentage: Days Present divided by Total Days
Punctuality Score: 100 minus (Late Arrivals multiplied by 5)
Performance Category: IF attendance >=95% then Excellent, IF attendance >=85% then Good, IF attendance >=75% then Average, else Poor
